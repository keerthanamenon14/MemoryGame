# 🎮 Memory Card Game

A **Memory Matching Game** built using **React, TypeScript, Redux Toolkit, and Tailwind CSS**.  
The game challenges users to match pairs of cards in a 4x4 grid with smooth interactions and clear feedback.

## 🚀 Setup & Run Instructions

### 1. Clone the repository
```bash
git clone https://github.com/keerthanamenon14/MemoryGame.git
cd memory-game

2. Install dependencies
npm install

3. Run the development server
npm run dev

4. Run tests
npm run test
